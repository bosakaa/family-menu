# 🍽️ Сімейне меню — GitHub Pages

Автоматична публікація тижневого меню через Claude + GitHub Pages.

---

## 🚀 Перше налаштування (один раз)

### 1. Створіть репозиторій на GitHub
- Зайдіть на [github.com](https://github.com) → **New repository**
- Назва: `family-menu` (або будь-яка)
- Visibility: **Public** *(обов'язково для безкоштовного Pages)*
- ✅ Add README

### 2. Клонуйте репозиторій на комп'ютер

```bash
git clone https://github.com/ВАШ-НІК/family-menu.git
cd family-menu
```

### 3. Скопіюйте файли з цього архіву в папку репозиторію

```
family-menu/
├── .github/
│   └── workflows/
│       └── deploy.yml       ← GitHub Actions (автодеплой)
├── index.html               ← поточне меню (замінюється щотижня)
├── HISTORY.md               ← архів меню по тижнях
├── update-menu.sh           ← скрипт оновлення (Mac/Linux)
├── update-menu.ps1          ← скрипт оновлення (Windows)
└── README.md
```

### 4. Зробіть скрипт виконуваним (Mac/Linux)

```bash
chmod +x update-menu.sh
```

### 5. Перший push

```bash
git add .
git commit -m "🚀 Початкове налаштування"
git push
```

### 6. Увімкніть GitHub Pages

GitHub → ваш репо → **Settings → Pages**
- Source: **GitHub Actions**
- Збережіть

Через 1-2 хвилини меню буде доступно за адресою:
**`https://ваш-нік.github.io/family-menu`**

---

## 📅 Щотижневий флоу (кожен тиждень, ~5 хвилин)

### Крок 1 — Генеруємо нове меню в Claude

Відкрийте Claude і напишіть:

```
Згенеруй меню на тиждень для нашої родини.
[Ваші побажання, зміни, коментарі]

Враховуй: що минулого тижня нам дуже сподобалось [страва],
а [інша страва] — пропусти цього тижня.
```

Claude поверне готовий HTML-файл — завантажте його.

### Крок 2 — Запустіть скрипт оновлення

**Mac / Linux:**
```bash
cd family-menu
./update-menu.sh ~/Downloads/menu-7-dniv.html "Тиждень 3: більше риби, без свинини"
```

**Windows (PowerShell):**
```powershell
cd family-menu
.\update-menu.ps1 -File "$env:USERPROFILE\Downloads\menu-7-dniv.html" -Message "Тиждень 3"
```

**Або просто запустіть без аргументів** — скрипт сам знайде останній завантажений файл:
```bash
./update-menu.sh
```

### Крок 3 — Готово!

Через 1-2 хвилини сайт оновиться автоматично. GitHub Actions сам задеплоїть.

## 💬 Корисні промпти для Claude

### Стандартне меню (копіюйте щотижня):
```
Згенеруй меню на 7 днів для нашої родини (2 дорослих + дитина 1р10м).
Умови: max 10 хв активної участі, техніка — духовка/аерогриль/прижимний гриль/плита.
Без гострого, без складних процесів.
Поверни повний HTML-файл як минулого разу.

[ЗМІНИ ЦЕЙ ТИЖДЕНЬ]:
- Хочемо більше: ___
- Менше або прибрати: ___
- Коментар: ___
```

### З урахуванням залишків:
```
Є залишки з минулого тижня: [список продуктів].
Побудуй меню так, щоб їх використати на початку тижня.
```

### Святковий тиждень:
```
На вихідних планується [свято/гості].
Зроби субота/неділя трохи святковішими, але без складних рецептів.
```

## ❓ Troubleshooting

**`git push` просить логін/пароль?**
Налаштуйте SSH або Personal Access Token:
- [Інструкція GitHub](https://docs.github.com/en/authentication)

**Pages не оновлюється?**
Перевірте: GitHub → репо → **Actions** → чи пройшов workflow без помилок.

**Скрипт не запускається (Mac)?**
```bash
chmod +x update-menu.sh
# Або якщо блокує macOS:
xattr -d com.apple.quarantine update-menu.sh
```
